# CLI
## Version Numbers
Version.Major.Minor.Bugfix
- a = alpha
- b = beta
- c = concept
- rc = Release Candidate
## Compilation
### Windows
1. Open PowerShell
2. Get Python 3.13 </br>
`winget install Python.Python.3.13`
3. Create a folder called "CLI"
4. Inside CLI, make a folder called "sys"
5. Inside sys, put cli.py
6. Move CLI.bat to CLI
